Seattle Gentrification Spatial Analysis (QGIS)
Overview

This project analyzes patterns of gentrification and transit accessibility in Seattle using spatial analysis techniques in QGIS. The project focuses on how light rail investment relates to low-income communities and displacement risk across Seattle neighborhoods.

Tools Used
QGIS
ACS 2020 5-Year Census Data
Seattle GeoData
Spatial Analysis
Choropleth Mapping
Project Goals
Identify low-income census tracts in Seattle
Analyze proximity to light rail stations
Examine relationships between transit investment and gentrification
Visualize spatial inequality using GIS mapping techniques
Methods
Joined ACS income data to Seattle census tract shapefiles using GEOID fields
Performed spatial analysis including:
Attribute queries
Multi-ring buffer analysis (0.25–1 mile)
Spatial clipping
Choropleth visualization
Compared transit accessibility across income levels
Key Findings

The analysis found that many lower-income neighborhoods have reduced proximity to light rail infrastructure, while higher-income neighborhoods tend to have greater transit access. Areas experiencing rapid investment also showed signs of increased displacement risk.
